### 1. 数据源管理页面

#### 布局结构

```
┌─────────────────────────────────────────────────────────────┐
│  数据源管理                                    [+ 新建数据源]  │
├─────────────────────────────────────────────────────────────┤
│                                                               │
│  ┌─────────────────────────────────────────────────────────┐│
│  │  🔍 搜索框                          [MySQL ▼] [全部 ▼]  ││
│  └─────────────────────────────────────────────────────────┘│
│                                                               │
│  ┌─────────────────────────────────────────────────────────┐│
│  │  数据源列表 (表格)                                        ││
│  │  ┌──────┬────────┬──────┬──────┬──────┬────────────┐   ││
│  │  │ 名称 │ 类型   │ 地址 │ 端口 │ 状态 │ 操作       │   ││
│  │  ├──────┼────────┼──────┼──────┼──────┼────────────┤   ││
│  │  │ MySQL│ MySQL  │ ... │ 3306 │ ✓   │ 测试 编辑  │   ││
│  │  │ ES-1 │ ES     │ ... │ 9200 │ ✓   │ 测试 编辑  │   ││
│  │  └──────┴────────┴──────┴──────┴──────┴────────────┘   ││
│  └─────────────────────────────────────────────────────────┘│
│                                                               │
└─────────────────────────────────────────────────────────────┘
```

#### 组件说明

**顶部操作栏**:
- 标题: "数据源管理" (18px, 粗体)
- 新建按钮: 主色按钮，右上角

**搜索和筛选区**:
- 搜索框: 左侧，占 60% 宽度
- 类型筛选: 下拉框 (MySQL / Elasticsearch / 全部)
- 状态筛选: 下拉框 (已连接 / 未连接 / 全部)

**数据源列表**:
- 表格形式展示
- 列: 名称、类型、地址、端口、状态、操作
- 状态: ✓ 绿色 / ✗ 红色
- 操作: 测试连接、编辑、删除
